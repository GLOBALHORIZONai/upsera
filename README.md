# Upsera — منصة التسويق الذاتية بالذكاء الاصطناعي
**by GLOBAL HORIZON INC.**

هذا المشروع يحتوي الآن على:
- **الواجهة** (`index.html`) — تعمل بالكامل في المتصفح، متعددة اللغات، دارك/لايت.
- **الخادم الآمن** (`api/analyze.js`) — دالة Serverless تستدعي Claude API بأمان (المفتاح لا يظهر للمتصفح).

---

## بنية المشروع

```
upsera-app/
├── index.html        ← الواجهة (Frontend)
├── api/
│   └── analyze.js    ← الخادم: يستدعي Claude بأمان
├── vercel.json       ← إعداد Vercel
├── package.json      ← إعداد Node
└── README.md         ← هذا الملف
```

---

## كيف يعمل (التدفق)

```
المستخدم يكتب نشاطه في الواجهة
        ↓
الواجهة ترسل الطلب إلى /api/analyze
        ↓
الخادم (يحفظ المفتاح بأمان) يستدعي Claude
        ↓
Claude يحلّل ويعيد: فهم النشاط + توصية + خطوات
        ↓
الواجهة تعرض التحليل في شاشة المراجعة
```

> ملاحظة مهمة: قبل النشر، الواجهة تعمل لكنها تعرض **تحليلاً تجريبياً** (لأن الخادم غير متصل بعد). بعد النشر على Vercel مع المفتاح، تحصل على **تحليل حقيقي** من Claude.

## النشر على Vercel — خطوة بخطوة

### الخطوة 1: جهّز حساب Vercel
1. ادخل https://vercel.com وسجّل دخولك (يمكن بحساب GitHub).

### الخطوة 2: ارفع المشروع
**الطريقة الأسهل (بدون GitHub):**
1. ثبّت أداة Vercel على جهازك: في الـ Terminal اكتب
   ```
   npm i -g vercel
   ```
2. ادخل مجلد المشروع:
   ```
   cd upsera-app
   ```
3. شغّل:
   ```
   vercel
   ```
   اتبع الأسئلة (اسم المشروع، إلخ). سينشر نسخة تجريبية ويعطيك رابطاً.

**أو عبر GitHub:**
1. ارفع مجلد `upsera-app` إلى مستودع GitHub.
2. في Vercel: New Project → استورد المستودع → Deploy.

### الخطوة 3: أضف مفتاح Claude API (الأهم)
1. احصل على المفتاح من https://console.anthropic.com (لديك مفتاح GLOBAL-HORIZON-AGENT).
2. في Vercel: افتح مشروعك → **Settings** → **Environment Variables**.
3. أضف متغيراً:
   - **Name:** `ANTHROPIC_API_KEY`
   - **Value:** (الصق مفتاحك هنا)
4. احفظ.

### الخطوة 4: أعد النشر
بعد إضافة المفتاح، أعد النشر ليأخذ الإعداد الجديد:
```
vercel --prod
```
أو من لوحة Vercel: Deployments → Redeploy.

### الخطوة 5: جرّب
افتح رابط موقعك → اختر مساراً → اكتب وصف نشاطك → اضغط "تابع".
ستحصل على تحليل حقيقي من Claude في شاشة المراجعة.

## ربط الدومين upsera.ai
1. في Vercel: Settings → **Domains** → أضف `upsera.ai`.
2. في لوحة مزوّد الدومين، أضف سجلات DNS التي يعطيك إياها Vercel.
3. انتظر انتشار DNS (دقائق إلى ساعات).

---

## الأمان
- المفتاح (`ANTHROPIC_API_KEY`) يُحفظ في إعدادات Vercel فقط، **ولا يظهر أبداً في الواجهة** أو للمتصفح.
- لا تضع المفتاح في `index.html` أو أي ملف يُرفع للمستودع العام إطلاقاً.
